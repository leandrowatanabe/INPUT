Inline CSS

Uma das formas de se utilizar o CSS é dentro do próprio HTML de forma inline

Outra forma é declarando um folha de estilos na head do HTML utilizando seletores. Os seletores podem se Tags, Nome de Classes, ou IDs. O uso de IDs tem um carater semântico de exclusividade, quando existe a necessidade de aplicar o mesmo estilo para mais de um elemento separado se usa as Classes.

Além disso é possível importar uma folha de estilo externar que aplica as regras da mesma forma. Essa importação é feita como um link externo ao arquivo de estilização.

Hierarquicamente, a preferência é interna, então a inline tem prioridade máxima na estilização. Logo em seguida vem a folha declarada interna ao HTML e por último o arquivo externo.

Comentarios no CSS

Nas folhas de estilização é possível fazer comentários, tanto dentro da tag declarada dentro do HTML quanto no arquivo externo.

CSS padrões

Cada navegador possuí um conjunto de estilizações padrões iniciais. Para garantir a homogeneidade das páginas é comum utilizar um css reset para padronizar em todos os navegadores.
