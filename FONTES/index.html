Tipografia e tipos de fontes


Quando estamos nos referindo de fontes, elas podem ser classificadas em dois principais, principalmente entre fontes Serifadas ou fontes Não Serifadas.


Serifadas

Não Serifadas


Além disso, existem mais três tipos que fazem parte compões as cinco famílias principais, a Monospace, Cursive, Fantasy. O Monospace separa espaços iguais para cada caracter. A Cursive é uma que imita a forma de escrita a mão. As do tipo Fantasy são as que possuem detalhes mais diferentes.

Fontes no CSS

Para utilizar fontes o navegador acessa as fontes que existem dentro do computador do usuário. É prática comum adicionar às declarações das fontes casos de retorno, que são utilizadas quando o navegador do usuário não possuí a fonte desejada, mas a estilização já prevê uma fonte substituta.

Carregando fontes externas

No CSS é possívle importar fontes declarando suas características e adicionando uma URL. A partir dessas declarações é possível acessar elas de qualquer parte do arquivo CSS.
