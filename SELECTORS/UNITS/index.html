Seletores CSS


A principal forma de selecionar um elemento em CSS é através da tag, outra forma é através das classes e, por último, através do ID. Podemos aplicar o mesmo conjunto de estilos agrupando em blocos. O CSS busca aplicar o estilo em todos os elementos que foram chamados, isso quer dizer que quando chamamos por tag, o CSS vai aplicar o estilo em todas as tags com o memso nome, no caso de classes, o CSS vai aplciar em todos os elementos com a classe declarada e assim também funciona para o ID.

Selecionando elementos especificos


Em CSS, é possível especificar qual é o elemento queremos aplicar a estilização. Isso quer dizer que podemos especificar a nível de tag, classe e ID de forma conjunta contatenando esses elementos.

Isso possibilita um nível de granularidade e especificidade que aumenta o controle que temos com relação a aplicação de estilos.

Descendant selector


Dentro do CSS também é possível aninhar seletores, no caso de tags, quando separados por espaço no CSS, é possível identificar um elemento específico que obedece à esse critério.

E daí, o CSS aplica a estilização apenas no elemento especificado pelo aninhamento. Isso funciona para multiplas classes também.

Child selector


Outra possibilidade é especificar com o caracter ">" o elemento diretamente filho, aplicando uma certa hierarquia na estilização.

Assim como o aninhamento, o CSS não aplica a estilização aos elementos que não obedecem a esse critério.

Sibiling selector


Outra possibilidade é especificar com o caracter "+" ou "~". Neste caso, o caracter "+" especifica que apenas o primeiro elemento após um seletor, de um conjunto de seletores, vai ser estilziado. Já o caracter "~" especifica que todos os seletores irmãos, filhos de um mesmo seletor, serão estilizados.

Universal selector


No CSS existe um seletor que é chamado de universal que é chamado com o caracter "*" que basicamente aplica a estilização em todos os elementos subsequentes. Isso quer dizer que se aplica dentro de um Div, todos os elementos dentro daquel div serão estilizados
