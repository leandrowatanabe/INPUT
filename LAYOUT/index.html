Tipos de elementos


Quando se trata de composição de layouts existem dois tipos de elementos, elementos do tipo Block e elementos do tipo inline. Os elementos do tipo Block eles variam sua largura conforme existe a disponibilidade no espaço fornecido pelo navegador. Já os elementos inline ocupam apenas a largura do conteúdo. Outra diferença que existe é que não é possível controlar o espaçamento vertical e nem a altura dos elementos inline.

Quando estamos falando de flexbox, estamos falando de um conjunto de elementos internos à um elemento que engloba esses elementos. Esse elemento englobador determina o posicionamentos dos itens internos a ele.

Alinhamento x e y

1
1

Para controlar o alinhamento dos elementos usamos o Justify-content e o Align-items. Por padrão Align-items recebe Strech, que significa ocupar o container na direção escolhida e Justify-content recebe flex-start que significa no início do container. Dependendo da direção, o controle dos eixos se invertem.
