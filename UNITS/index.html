CSS units - Absolutas avs Relativas


Quando trabalhos com tamanhos e espaçamentos utilizamos unidades de medidas. No CSS existem dois tipos de unidades de medidas, as Absolutas e as Relativas. As absolutas são unidades independentes, as três mais conhecidas são pixel(px), points(pt) e centimetros (cm). As relativas são unidades que dependem de outra medida, as mais conhecidas são porcentagem (%), fracionário (fr), em e rem.

Unidade de medida - porcentagem


A unidade de medida porcentual baseia o seu valor na propriedade parental logo acima dela. Ou seja, uma div com tamanho porcentual vai buscar se basear seu valor em uma div parental.

Unidade de medida - em


A unidade de medida em baseia seu valor na font-size do elemento parental a ele. Lembrando que ele obedece a hierarquia, então ele sempre vai ter como base o elemento parental

Unidade de medida - rem


A unidade de medida em baseia seu valor na font-size do elemento root. Em outras palavras, a não ser que se declare um elemento root que não seja o padrão, a unidade de medida rem vai se basear na font-size do HTML.
