Controlando o tamanho dos elementos


Em css é comum utilizar o box model para controlar os tamanhos dos elementos. O tamanho que ele apresenta na tela é influenciado pela margem, pelo preenchimento e pela borda. Quando estamos trabalhando com um tamanho de conteúdo fixo, então a propriedade "box-sizing: content-box" garante que as dimensões declaradas são do tamanho do conteúdo do elemento. Já a propriedade "box-sizing: border-box" garante que as dimensões são as declaradas na sua totalidade.

Dentro dos elementos, o atribuito overflow é o que controla a exsitência ou não da barra de rolagem. Essa barra é utilizada quando o conteúdo dos elementos ocupam mais espaço que os próprios elementos.

Entendendo posicionamento de elementos


Podemos posicionar elementos através do atributo "position" que pode receber os valores absolute, relative e fixed. Os valores relative e fixed podem ser relacionados a outros elementos obedecendo uma hierarquia, já o absolute coloca em uma posição fixa da página.

Entendendo posicionamento de elementos em 3d


Em se tratando de elementos de um layout web, é possível criar posicionamentos relativos à profundiade. Isso quer dizer que é possível posicionar elementos em cima ou baixo da perspectiva do usuário. Para controlar o posicionamento existe o atributo "z-index".

Entendendo media queries


Em resposta aos mais diversos tamanhos de mídias que existem atualmente por conta de tablets, celulares e computadores, as páginas tem utilizado as media queries. São funcionalidades do CSS que aplicam atributos de CSS a seletores específicos de acordo com uma condição, geralmente o tamanho da mídia.
